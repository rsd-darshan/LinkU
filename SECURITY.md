# Security policy

## Supported versions

Security fixes are applied on the default branch (`main`) for this repository.

## Reporting a vulnerability

Please report security issues privately (do not open a public issue with exploit details).

- **Preferred:** GitHub [private vulnerability reporting](https://github.com/rsd-darshan/LinkU/security/advisories/new) for this repository, if enabled.
- **Alternative:** Contact the repository owner via the email on their GitHub profile.

Include steps to reproduce, affected components (e.g. API route path), and severity assessment when possible.
